REQUIREMENTS
-------------------------------------------------------------

To edit this template, you will need:
- A modern code editor (e.g., Visual Studio Code, Sublime Text)
- A modern browser (Chrome, Firefox, Edge) for preview/testing
- Basic HTML/CSS/JS knowledge

No build tools are required. Everything is pure HTML + Tailwind CSS CDN.

-------------------------------------------------------------
INSTRUCTIONS
-------------------------------------------------------------

1. Unzip the downloaded file.
2. Open `index.html` in your browser to view the homepage.
3. Edit content directly in the HTML files using any code editor.
4. Replace images in the `assets/images/` folder as needed.
5. Update colors, text, and layout by editing HTML + Tailwind classes.

-------------------------------------------------------------
CUSTOMIZATION
-------------------------------------------------------------

- COLORS:
  Use Tailwind CSS utility classes (e.g., `bg-orange-500`, `text-white`) to
  change colors.

- FONTS:
  The template uses Google Fonts. You can edit or replace them in the
  `<head>` section of the HTML files.

- IMAGES:
  All demo images are located in `assets/images/`. Replace them as needed.
  Ensure images are optimized for web.
