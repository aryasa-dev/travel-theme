🚀 Getting Started

To use this template:

  • Unzip the downloaded folder
  • Open index.html in your browser
  • Edit content using any code editor (e.g. VS Code)

📦 Included Files

  • index.html – Homepage
  • assets/css/style.css – Main stylesheet
  • assets/js/main.js – JavaScript logic
  • assets/images/ – Images used in the template

⚙️ Customization

1. Colors

Colors are defined in Tailwind utility classes (e.g. bg-[#0A1E3F], text-orange-500). You can change them directly in the HTML or customize Tailwind config if using Tailwind CLI.

2. Images

All images are located in assets/images/. Replace them with your own if needed. Use optimized WebP or JPG formats.

3. Fonts

This template uses Google Fonts – change or add fonts by editing <link> tags in the <head> section.

🧠 JavaScript Features

  • Sticky Navbar
  • Smooth Scroll
  • FAQ Toggle
  • Animated on Scroll (AOS library)

JavaScript files are located in assets/js/. Main functions are inside main.js.

📱 Responsive Design

This template is fully responsive and tested across modern browsers and screen sizes using Tailwind CSS utility classes.
